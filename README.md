# Assembler — Курсовая работа

Двухпроходной ассемблер, реализованный на C++.

## Описание

Программа читает исходный файл на языке ассемблера, выполняет два прохода:

1. **Первый проход** — обход всех строк исходного кода, назначение адресов инструкциям и формирование таблицы символов (меток).
2. **Второй проход** — генерация машинного кода с подстановкой реальных адресов меток.

По результатам сборки создаётся файл листинга (`.lst`) с адресами, hex-кодами и исходными строками.

## Система команд

| Команда | Формат | Описание |
|---------|--------|----------|
| MOV | `MOV reg, reg/imm` | Пересылка значения |
| ADD | `ADD reg, reg/imm` | Сложение |
| SUB | `SUB reg, reg/imm` | Вычитание |
| MUL | `MUL reg, reg/imm` | Умножение |
| DIV | `DIV reg, reg/imm` | Деление |
| AND | `AND reg, reg/imm` | Побитовое И |
| OR  | `OR  reg, reg/imm` | Побитовое ИЛИ |
| XOR | `XOR reg, reg/imm` | Побитовое исключающее ИЛИ |
| NOT | `NOT reg` | Побитовое НЕ |
| CMP | `CMP reg, reg/imm` | Сравнение (устанавливает флаги) |
| JMP | `JMP label` | Безусловный переход |
| JZ  | `JZ  label` | Переход если ноль |
| JNZ | `JNZ label` | Переход если не ноль |
| JG  | `JG  label` | Переход если больше |
| JL  | `JL  label` | Переход если меньше |
| PUSH | `PUSH reg/imm` | Запись в стек |
| POP  | `POP  reg`    | Чтение из стека |
| CALL | `CALL label`  | Вызов подпрограммы |
| RET  | `RET`         | Возврат из подпрограммы |
| IN   | `IN  reg`     | Ввод данных |
| OUT  | `OUT reg`     | Вывод данных |
| NOP  | `NOP`         | Нет операции |
| HLT  | `HLT`         | Остановка |

### Директивы данных

| Директива | Описание |
|-----------|----------|
| `DB val[, val...]` | Определение байта(ов) |
| `DW val[, val...]` | Определение слова(ов) (2 байта) |

### Регистры

`AX`, `BX`, `CX`, `DX`, `SP`, `BP`

### Синтаксис

```asm
; Комментарий
МЕТКА:  КОМАНДА  операнд1, операнд2
```

## Сборка

```bash
make
```

или вручную:

```bash
g++ -std=c++11 -Wall -O2 -o assembler assembler.cpp
```

## Использование

```bash
./assembler <входной_файл.asm> [выходной_файл.lst]
```

Пример:

```bash
./assembler example.asm example.lst
```

## Пример программы

Файл `example.asm` содержит программу вычисления суммы 1+2+...+10:

```asm
        MOV  AX, 0      ; сумма = 0
        MOV  CX, 10     ; счётчик = 10
LOOP:
        ADD  AX, CX     ; сумма += CX
        SUB  CX, 1      ; CX--
        CMP  CX, 0
        JNZ  LOOP       ; если CX != 0, повторить
        OUT  AX         ; вывести результат (55)
        HLT
```

## Структура листинга

```
LINE  ADDR  HEX            SOURCE
----------------------------------------------------------------------
1     0000  11 00 00 00    MOV  AX, 0
2     0004  11 02 0A 00    MOV  CX, 10
...
```

## Формат машинного кода

Каждая инструкция кодируется следующим образом:

| Байт 0 | Байт 1 | Байт 2 | Байт 3 |
|--------|--------|--------|--------|
| Код операции | Регистр назначения | Регистр/данные (мл.) | Данные (ст.) |

Коды регистров: AX=0x00, BX=0x01, CX=0x02, DX=0x03, SP=0x04, BP=0x05
